  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get MeetMates up and running in minutes
  </p>

  <Steps>
    <Step title="Clone the Repository">
      First, clone the MeetMates repository to your local machine:

      ```bash Terminal
      git clone https://github.com/divyanshraj0408/meetmates.git
      cd meetmates
      ```
    </Step>

    <Step title="Install Dependencies">
      Install dependencies for both frontend and backend:

      ```bash Terminal
      cd frontend && npm install
      cd ../backend && npm install
      ```
    </Step>

    <Step title="Configure Environment">
      Set up your environment variables for both the frontend and backend:

      ```bash frontend/.env
      VITE_BACKEND_URL=http://localhost:3001
      VITE_GOOGLE_CLIENT_ID=your_google_client_id
      ```

      ```bash backend/.env
      MONGODB_URI=your_mongodb_uri
      JWT_SECRET=your_jwt_secret
      CORS_ORIGIN=http://localhost:5173
      PORT=3001
      ```

      <Note>
        You'll need to obtain a Google OAuth Client ID from the [Google Cloud Console](https://console.cloud.google.com/).
      </Note>
    </Step>

    <Step title="Start the Application">
      Launch both the frontend and backend servers:

      ```bash Terminal
      # In the backend directory
      npm start

      # In a new terminal, in the frontend directory
      npm run dev
      ```

      Your application will be running at `http://localhost:5173`!
    </Step>
  </Steps>
</div>
